jquery.simpleImageDiff

Responsive image diff tool Works in all modern browsers include IE >= 9

Install

    bower install jquery.simpleImageDiff --save

Simple layout

responsive 

    <div data-diff="1">
        <img src="images/1.jpg" />
        <img src="images/2.jpg" />
    </div>

Initialization

    $('[data-diff]').simpleImageDiff();
